# archive/termios の内容

このディレクトリは、`termios` を使った DDSM シリアル通信の旧コードを退避している場所です。

## ファイル一覧

- `motor_driver.c`  
  指定した RPM を4輪（ID 1〜4）へ順番に送信し、応答を確認する実験用ドライバ。

- `mv_motor.c`  
  1輪（指定 ID）に対して速度コマンドを送受信し、CRC なども確認する単体通信テスト。

- `mv_same.c`  
  同じ RPM を4輪へ一括送信するテストコード（`motor_driver.c` と近い用途）。

- `receive_data.c`  
  フィードバック要求コマンドを送って、10バイト応答を受信・表示する確認用コード。

## メモ

- ここは `archive` 配下のため、現行実装というより保管目的です。
- いずれのコードもデバイスパスは主に `/dev/ddsm` を前提にしています。
