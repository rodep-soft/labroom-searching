# archive/set_motor_id の内容

このディレクトリは、モータID変更用の旧ユーティリティコードを退避している場所です。

## ファイル一覧

- `set_id.cpp`  
  Boost.Asio のシリアル通信で `/dev/ttyUSB0` にコマンドを送信し、モータIDを変更するためのツール。

## 動作概要

- 実行引数で新しい ID を指定する（例: `./set_motor_id 1`）。
- 送信コマンドは `0xAA 0x55 0x53 <new_id> ...` 形式。
- 送信後、モータの電源を入れ直すことで新 ID を有効化する想定。

## メモ

- ここは `archive` 配下のため、現行実装というより保管目的です。
- ポート名はコード内で `/dev/ttyUSB0` 固定になっています。
- ビルドには Boost ライブラリが必要で、`g++ set_id.cpp -o set_id -lboost_system` などで可能です。
- **注意**: 実行前にモータの電源を切ることを推奨します。またちゃんと動くかわかりません。実際はROXのモーターを使ったためmotor_idを設定してません。
