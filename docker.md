# Dockerの使い方
Dockerは、アプリケーションをコンテナとしてパッケージ化し、どこでも一貫して動作させるためのプラットフォームです。以下に、Dockerの基本的な使い方を説明します。


```bash
docker compose build
docker compose up -d
docker compose ps
```

 ## docker compose build
イメージを作る
